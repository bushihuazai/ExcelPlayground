# 01 — LAMBDA 基础：定义可复用的自定义函数

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

`LAMBDA` 是 Excel 函数式编程的基石。它允许你把一段计算逻辑封装成**可复用的函数**，既能在单元格中内联使用，也能通过"名称管理器"注册为全局函数。

---

## 示例 1 ⭐ — 内联调用：计算矩形面积

```excel
=LAMBDA(长, 宽, 长*宽)(A2, B2)
```

| 参数 | 说明 |
|------|------|
| `长` | 矩形的长 |
| `宽` | 矩形的宽 |

**说明**：`LAMBDA(参数1, 参数2, 表达式)(实参1, 实参2)` 是最基本的写法——先定义，再在末尾 `()` 传入实参立即调用。

---

## 示例 2 ⭐⭐ — 命名函数：安全除法 `SAFE_DIV`

在 **名称管理器** 中新建名称 `SAFE_DIV`，引用至：

```excel
=LAMBDA(a, b, IF(b=0, "", a/b))
```

在单元格中使用：

```excel
=SAFE_DIV(A2, B2)
```

**说明**：将 LAMBDA 注册到名称管理器后，它就变成了一个"用户自定义函数"，团队所有人都可以像内置函数一样调用。

> ⚠️ **注意**：名称管理器中的 LAMBDA 不需要尾部 `()` 调用，因为调用发生在单元格里。

---

## 示例 3 ⭐⭐⭐ — 折后价计算（含下限保护）

```excel
=LAMBDA(price, discount,
  LET(
    result, price * (1 - discount),
    MAX(0, result)
  )
)(A2, B2)
```

**说明**：当折扣比例输入错误（如 >100%）时，`MAX(0, result)` 确保返回值不会出现负数。LAMBDA 内部可嵌套 `LET`、`IF` 等任何函数。

---

## 示例 4 ⭐⭐⭐⭐ — 递归：文本重复 N 次

在名称管理器中定义 `REPEAT_TEXT`：

```excel
=LAMBDA(txt, n,
  IF(n<=0, "",
    txt & REPEAT_TEXT(txt, n-1)
  )
)
```

使用：

```excel
=REPEAT_TEXT("★", 5)
```

结果：`★★★★★`

**说明**：LAMBDA 支持**递归调用自身**（必须通过命名函数），适用于文本拼接、树形展开等场景。

> ⚠️ **注意**：递归深度受 Excel 调用栈限制，通常安全范围为 **≤128 层**。超过可能报 `#NUM!` 错误。

---

## 应用注意点

| 要点 | 说明 |
|------|------|
| 参数命名 | 使用有意义的名称（如 `price` 而非 `x`），提升可读性 |
| 内联 vs 命名 | 一次性计算用内联；多次复用或团队共享用命名 |
| 递归限制 | 递归深度通常 ≤128 层，处理大数据时优先考虑 `REDUCE` / `SCAN` |
| 调试技巧 | 先在单元格中用内联 LAMBDA 测试逻辑，确认无误后再注册为命名函数 |
| 兼容性 | LAMBDA 仅在 Microsoft 365 / Excel 2021+ 中受支持，旧版本无法使用 |
