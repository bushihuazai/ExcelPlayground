# 11 — FILTER 条件筛选：声明式数据过滤

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

`FILTER` 根据条件从数组中提取符合要求的行（或列），返回动态数组。它替代了传统的高级筛选和复杂的 INDEX/MATCH 数组公式。

语法：`=FILTER(array, include, [if_empty])`

- `include`：与 array 同行的布尔数组（TRUE/FALSE）
- `if_empty`：全部不满足时的返回值

---

## 示例 1 ⭐ — 单条件筛选

```excel
=FILTER(A2:E100, C2:C100="华东", "无数据")
```

**说明**：筛选 C 列为"华东"的所有行。第三参数 `"无数据"` 防止结果为空时报 `#CALC!` 错误。

---

## 示例 2 ⭐⭐ — 多条件 AND（同时满足）

```excel
=FILTER(A2:E100,
  (C2:C100="华东") * (E2:E100>10000),
  "无匹配"
)
```

**说明**：`*` 号表示逻辑 AND（两个条件都为 TRUE 时乘积为 1）。筛选华东区**且**销售额 >10000 的记录。

---

## 示例 3 ⭐⭐⭐ — 多条件 OR（满足其一）

```excel
=FILTER(A2:E100,
  (C2:C100="华东") + (C2:C100="华南"),
  "无匹配"
)
```

**说明**：`+` 号表示逻辑 OR（任一条件为 TRUE 时和 ≥1）。筛选华东**或**华南的记录。

> ⚠️ 使用 `+` 做 OR 时，结果可能大于 1。FILTER 内部将 >0 视为 TRUE，所以逻辑正确。

---

## 示例 4 ⭐⭐⭐⭐ — 动态条件：下拉框驱动筛选

假设 H1 为区域下拉框，H2 为最低金额输入框。

```excel
=FILTER(A2:E100,
  (IF(H1="全部", TRUE, C2:C100=H1)) *
  (E2:E100>=H2),
  "无匹配记录"
)
```

**说明**：当 H1 选择"全部"时忽略区域条件，否则按选定区域筛选。这是纯公式实现的**交互式仪表板**基础。

---

## 示例 5 ⭐⭐⭐⭐⭐ — FILTER 嵌套：Top N 筛选

找出销售额排名前 5 的记录：

```excel
=LET(
  data,    A2:E100,
  amount,  INDEX(data,,5),
  valid,   FILTER(data, ISNUMBER(amount)),
  sorted,  SORT(valid, 5, -1),
  top5,    FILTER(sorted,
             INDEX(sorted,,5) >= LARGE(INDEX(sorted,,5), 5)
           ),
  top5
)
```

**说明**：

1. `FILTER` 去掉非数值行
2. `SORT` 按第 5 列降序
3. 再次 `FILTER` 取销售额 ≥ 第 5 大值的行

> ⚠️ 如果第 5 名有并列，结果可能超过 5 行。如需严格取 5 行，可改用 `INDEX + SEQUENCE`。

---

## 应用注意点

| 要点 | 说明 |
|------|------|
| if_empty 参数 | 强烈建议始终提供第三参数，避免无结果时报 `#CALC!` |
| AND / OR | 用 `*` 表示 AND，用 `+` 表示 OR；可混合使用 |
| 结果是动态数组 | 结果会自动溢出，确保下方有足够空间 |
| 列筛选 | FILTER 也可以按列筛选：`=FILTER(A1:E1, A2:E2>0)` |
| 与 SORT 搭配 | `SORT(FILTER(...))` 是最常见的组合，先过滤再排序 |
