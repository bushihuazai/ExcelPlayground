# 07 — SCAN 过程可视化：展示每一步的中间结果

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

`SCAN` 与 `REDUCE` 的计算逻辑完全相同，区别在于：

- **REDUCE** → 只返回最终结果（标量）
- **SCAN** → 返回**每一步的累加器值**（数组）

语法：`=SCAN(初始值, array, LAMBDA(acc, x, 表达式))`

---

## 示例 1 ⭐ — 运行累计和（Running Total）

```excel
=SCAN(0, A2:A20, LAMBDA(acc, x, acc + x))
```

**说明**：结果是 19 个数值，分别对应 A2 的累计和、A2:A3 的累计和 … 一直到 A2:A20 的总和。在财务报表和库存管理中极为常用。

---

## 示例 2 ⭐⭐ — 账户余额轨迹

```excel
=SCAN(10000, B2:B50, LAMBDA(余额, 现金流, 余额 + 现金流))
```

| 参数 | 说明 |
|------|------|
| `10000` | 初始账户余额 |
| `B2:B50` | 每日收支（正为收入，负为支出） |

**说明**：每一行显示当日结余。配合条件格式，余额低于 0 时标红，可快速发现透支风险。

---

## 示例 3 ⭐⭐⭐ — 累计增长率曲线

```excel
=SCAN(1, C2:C13, LAMBDA(acc, rate, acc * (1 + rate)))
```

**说明**：

- 初始值 `1` 代表基准（100%）
- 每月乘以 `(1+当月收益率)`
- 结果是 12 个数值，展示投资净值走势

配合折线图即可绘制"净值曲线"。

---

## 示例 4 ⭐⭐⭐⭐ — 编号自动生成（按分组重置）

假设 A 列是部门名称，希望在同一部门内从 1 递增，换部门时重置。

```excel
=SCAN(0, SEQUENCE(ROWS(A2:A100)), LAMBDA(acc, i,
  IF(i=1, 1,
    IF(INDEX(A:A, i+1) = INDEX(A:A, i), acc+1, 1)
  )
))
```

**说明**：

- `i` 是行索引
- 如果当前行与上一行的部门相同，计数 +1；否则重置为 1
- 前提：数据已按部门排序

> ⚠️ 此公式假设数据从第 2 行开始，`INDEX(A:A, i+1)` 对应实际行号。

## 示例 5 ⭐⭐⭐⭐⭐ — 动态止损线计算（最高水位回撤）

```excel
=LET(
  prices, D2:D252,
  highwater, SCAN(INDEX(prices,1), prices, LAMBDA(hw, p, MAX(hw, p))),
  drawdown, (prices - highwater) / highwater,
  HSTACK(prices, highwater, drawdown)
)
```

| 列 | 含义 |
|----|------|
| 第 1 列 | 当日价格 |
| 第 2 列 | 历史最高价（High Water Mark） |
| 第 3 列 | 回撤幅度（负值，越小回撤越大） |

**说明**：`SCAN` 维护一个"最高水位"，每步取当前价格与历史最高的较大值。回撤幅度 = (当前价 − 最高价) / 最高价。此方案常用于基金风控和量化策略。

---

## 应用注意点

| 要点 | 说明 |
|------|------|
| 返回值长度 | SCAN 返回数组长度 = 输入数组长度（不包含初始值本身） |
| 调试利器 | 将 REDUCE 改成 SCAN 即可查看每步状态，排查逻辑错误 |
| 图表配合 | SCAN 的输出天然适合作为折线图数据源 |
| 初始值 | 初始值不出现在结果中，但会参与第一次计算 |
| 性能 | 与 REDUCE 相同，逐元素调用 LAMBDA，数据量大时注意性能 |
