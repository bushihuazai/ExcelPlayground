# 16 — 人力资源数据分析：员工统计与薪资洞察

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

HR 部门日常需要对员工花名册做多维度分析：人数统计、薪资分布、任期计算、离职率追踪等。本场景综合运用 `LET`、`UNIQUE`、`MAP`、`FILTER`、`SORT`、`SCAN`、`HSTACK`、`VSTACK` 等函数，在一个公式中完成从原始花名册到可交付分析报表的全流程。

假设数据区域 `A2:H500`，列依次为：**工号(A)、姓名(B)、部门(C)、职级(D)、入职日期(E)、薪资(F)、状态(G: 在职/离职)、离职日期(H)**。

---

## 示例 1 ⭐ — 各部门在职人数统计

```excel
=LET(
  data,   A2:H500,
  dept,   INDEX(data,,3),
  status, INDEX(data,,7),
  active, FILTER(dept, status="在职"),
  u,      SORT(UNIQUE(active)),
  cnt,    MAP(u, LAMBDA(d, ROWS(FILTER(active, active=d)))),
  header, {"部门","在职人数"},
  VSTACK(header, HSTACK(u, cnt))
)
```

**说明**：先 `FILTER` 筛选在职员工，再 `UNIQUE` 提取部门列表，`MAP` 逐部门计数。这是最基础的分组统计，等价于数据透视表中对部门字段做 COUNT。

## 示例 2 ⭐⭐ — 部门薪资概览（均值 / 中位数 / 最大值）

```excel
=LET(
  data,    A2:H500,
  dept,    INDEX(data,,3),
  salary,  INDEX(data,,6),
  status,  INDEX(data,,7),
  mask,    (status="在职") * ISNUMBER(salary),
  u,       SORT(UNIQUE(FILTER(dept, mask))),
  avgS,    MAP(u, LAMBDA(d, ROUND(AVERAGE(FILTER(salary, (dept=d)*mask)), 0))),
  medS,    MAP(u, LAMBDA(d, MEDIAN(FILTER(salary, (dept=d)*mask)))),
  maxS,    MAP(u, LAMBDA(d, MAX(FILTER(salary, (dept=d)*mask)))),
  header,  {"部门","平均薪资","中位数薪资","最高薪资"},
  body,    HSTACK(u, avgS, medS, maxS),
  VSTACK(header, SORT(body, 2, -1))
)
```

**说明**：

1. `mask` 同时过滤在职状态和有效薪资数据
2. `MAP` 为每个部门分别计算 AVERAGE、MEDIAN、MAX
3. 最终按平均薪资降序排列，快速识别高薪部门

> 备选：将 `MEDIAN` 替换为 `PERCENTILE.INC(..., 0.75)` 可查看 P75 薪资水平。

## 示例 3 ⭐⭐⭐ — 员工任期分布（分段统计）

```excel
=LET(
  data,    A2:H500,
  hire,    INDEX(data,,5),
  status,  INDEX(data,,7),
  today,   TODAY(),
  tenure,  FILTER((today - hire) / 365.25, status="在职"),
  breaks,  {0, 1, 3, 5, 10, 99},
  n,       COLUMNS(breaks) - 1,
  labels,  MAP(SEQUENCE(1, n), LAMBDA(i,
             IF(INDEX(breaks,,i+1)=99,
               INDEX(breaks,,i) & "年以上",
               INDEX(breaks,,i) & "~" & INDEX(breaks,,i+1) & "年"
             )
           )),
  counts,  MAP(SEQUENCE(1, n), LAMBDA(i,
             LET(
               lo, INDEX(breaks,,i),
               hi, INDEX(breaks,,i+1),
               SUM((tenure >= lo) * (tenure < hi))
             )
           )),
  pcts,    MAP(counts, LAMBDA(c, ROUND(c / SUM(counts) * 100, 1) & "%")),
  VSTACK(labels, counts, pcts)
)
```

**说明**：

1. 用 `(TODAY() - hire) / 365.25` 计算工龄（年）
2. `{0,1,3,5,10,99}` 定义分段断点：<1年、1~3年、3~5年、5~10年、10年以上
3. `MAP` 遍历区间计数并计算占比百分比

> 备选：修改 `breaks` 常量即可自定义分段区间，无需改公式结构。

## 示例 4 ⭐⭐⭐⭐ — 薪资等级分布矩阵（部门 × 职级）

```excel
=LET(
  data,    A2:H500,
  dept,    INDEX(data,,3),
  level,   INDEX(data,,4),
  salary,  INDEX(data,,6),
  status,  INDEX(data,,7),
  mask,    (status="在职") * ISNUMBER(salary),
  u_dept,  SORT(UNIQUE(FILTER(dept, mask))),
  u_level, SORT(UNIQUE(FILTER(level, mask))),
  n_d,     ROWS(u_dept),
  n_l,     ROWS(u_level),
  body,    MAKEARRAY(n_d, n_l, LAMBDA(r, c,
             LET(
               d, INDEX(u_dept, r),
               l, INDEX(u_level, c),
               vals, FILTER(salary, (dept=d)*(level=l)*mask, 0),
               IF(ROWS(vals)=1, IF(vals=0, "-", ROUND(AVERAGE(vals),0)),
                  ROUND(AVERAGE(vals), 0))
             )
           )),
  header,  HSTACK("部门 \ 职级", TOROW(u_level)),
  rows,    HSTACK(u_dept, body),
  VSTACK(header, rows)
)
```

**说明**：

1. 构建"部门 × 职级"二维交叉矩阵
2. `MAKEARRAY` 遍历每个部门-职级组合，计算平均薪资
3. 无数据的组合显示 `"-"`，避免空值干扰

> 备选：将 `AVERAGE` 替换为 `ROWS` 可切换为各部门-职级的人数矩阵。

## 示例 5 ⭐⭐⭐⭐⭐ — 完整 HR 仪表板：多维度综合分析报告

假设 Z1 为部门下拉框（含"全部"选项），Z2 为统计截止日期。

```excel
=LET(
  /* ① 数据源与动态筛选 */
  raw,      A2:H500,
  dept,     INDEX(raw,,3),
  level,    INDEX(raw,,4),
  hire,     INDEX(raw,,5),
  salary,   INDEX(raw,,6),
  status,   INDEX(raw,,7),
  leave_dt, INDEX(raw,,8),
  cutoff,   Z2,

  mask_dept, IF(Z1="全部", TRUE, dept=Z1),
  mask_active, (status="在职") * ISNUMBER(salary),
  active,   FILTER(raw, mask_dept * mask_active, ""),

  /* ② 基础指标 */
  a_salary, INDEX(active,,6),
  a_hire,   INDEX(active,,5),
  headcount, ROWS(active),
  avg_sal,   ROUND(AVERAGE(a_salary), 0),
  med_sal,   MEDIAN(a_salary),
  avg_tenure, ROUND(AVERAGE((cutoff - a_hire)/365.25), 1),

  /* ③ 离职率（过去 12 个月） */
  left_12m,  ROWS(FILTER(raw,
               (status="离职") *
               mask_dept *
               (leave_dt >= cutoff - 365) *
               (leave_dt <= cutoff),
             "")),
  turnover,  ROUND(left_12m / MAX(headcount, 1) * 100, 1),

  /* ④ 薪资四分位 */
  q1, PERCENTILE.INC(a_salary, 0.25),
  q3, PERCENTILE.INC(a_salary, 0.75),

  /* ⑤ 按职级分布 */
  a_level,   INDEX(active,,4),
  u_level,   SORT(UNIQUE(a_level)),
  lv_cnt,    MAP(u_level, LAMBDA(l, ROWS(FILTER(a_level, a_level=l)))),
  lv_avg,    MAP(u_level, LAMBDA(l, ROUND(AVERAGE(FILTER(a_salary, a_level=l)),0))),

  /* ⑥ 组装输出 */
  summary, VSTACK(
    {"【汇总指标】",""},
    HSTACK("在职人数", headcount),
    HSTACK("平均薪资", avg_sal),
    HSTACK("中位数薪资", med_sal),
    HSTACK("薪资 Q1 / Q3", q1 & " / " & q3),
    HSTACK("平均工龄(年)", avg_tenure),
    HSTACK("年离职率%", turnover & "%"),
    {"",""},
    {"【职级分布】","人数","平均薪资"},
    HSTACK(u_level, lv_cnt, lv_avg)
  ),

  summary
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| ① | 动态筛选（部门 + 在职状态） | FILTER + IF |
| ② | 基础 KPI（人数、薪资、工龄） | AVERAGE、MEDIAN、ROWS |
| ③ | 年离职率 | FILTER 按日期范围 + 百分比 |
| ④ | 薪资四分位 | PERCENTILE.INC |
| ⑤ | 职级分布明细 | UNIQUE + MAP |
| ⑥ | 组装报表 | VSTACK + HSTACK |

一个公式输出完整的 HR 分析仪表板——切换部门下拉框即刻刷新全部指标。

## 应用注意点

| 要点 | 说明 |
|------|------|
| 日期格式 | 确保入职/离职日期为 Excel 日期序列号，否则计算天数差会报错 |
| 空值处理 | 薪资列可能有空值，`ISNUMBER(salary)` 过滤非数值数据 |
| 大数据量 | 员工超 5,000 时 MAP+FILTER 嵌套可能变慢，可用 SUMIFS / COUNTIFS 替代 |
| 隐私合规 | 薪资数据敏感，输出报告时注意按权限控制访问 |
| 扩展建议 | 可将各步骤封装为命名 LAMBDA 函数（如 `HR_TURNOVER`），提升团队复用率 |
