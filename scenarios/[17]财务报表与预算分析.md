# 17 — 财务报表与预算分析：从明细到报表的纯公式方案

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

财务分析中，常需从交易明细生成损益表、预算对比、现金流追踪等报表。本场景展示如何用 `LET`、`FILTER`、`UNIQUE`、`MAP`、`REDUCE`、`SCAN`、`HSTACK`、`VSTACK` 等函数，在一个公式中完成财务数据的清洗、分类汇总与报表输出。

假设交易明细 `A2:G1000`，列依次为：**日期(A)、凭证号(B)、科目(C)、摘要(D)、借方(E)、贷方(F)、部门(G)**。

---

## 示例 1 ⭐ — 按科目汇总借贷发生额

```excel
=LET(
  data,    A2:G1000,
  acct,    INDEX(data,,3),
  debit,   INDEX(data,,5),
  credit,  INDEX(data,,6),
  valid,   ISNUMBER(debit) + ISNUMBER(credit),
  u,       SORT(UNIQUE(FILTER(acct, valid))),
  sumD,    MAP(u, LAMBDA(a, SUMPRODUCT((acct=a) * IF(ISNUMBER(debit), debit, 0)))),
  sumC,    MAP(u, LAMBDA(a, SUMPRODUCT((acct=a) * IF(ISNUMBER(credit), credit, 0)))),
  bal,     sumD - sumC,
  header,  {"科目","借方合计","贷方合计","余额"},
  VSTACK(header, SORT(HSTACK(u, sumD, sumC, bal), 4, -1))
)
```

**说明**：逐科目汇总借方和贷方发生额，计算余额 = 借方 − 贷方，按余额降序排列。这是财务报表最基础的科目余额表。

## 示例 2 ⭐⭐ — 月度收入费用趋势表

```excel
=LET(
  data,    A2:G1000,
  dates,   INDEX(data,,1),
  acct,    INDEX(data,,3),
  debit,   INDEX(data,,5),
  credit,  INDEX(data,,6),
  ym,      YEAR(dates) * 100 + MONTH(dates),

  /* 收入类科目以"6"开头，费用类以"5"开头 */
  is_rev,  LEFT(acct, 1) = "6",
  is_exp,  LEFT(acct, 1) = "5",
  u_month, SORT(UNIQUE(FILTER(ym, ISNUMBER(dates)))),

  rev_by_m, MAP(u_month, LAMBDA(m,
    SUMPRODUCT((ym=m) * is_rev * IF(ISNUMBER(credit), credit, 0))
  )),
  exp_by_m, MAP(u_month, LAMBDA(m,
    SUMPRODUCT((ym=m) * is_exp * IF(ISNUMBER(debit), debit, 0))
  )),
  profit,  rev_by_m - exp_by_m,

  header, {"月份","收入","费用","利润"},
  VSTACK(header, HSTACK(u_month, rev_by_m, exp_by_m, profit))
)
```

**说明**：

1. `YEAR(dates)*100+MONTH(dates)` 将日期转为 YYYYMM 格式的月份编号
2. 通过科目编号首位字符区分收入类（6 开头）和费用类（5 开头）
3. 按月汇总后计算利润 = 收入 − 费用

> 备选：如果科目编号规则不同，只需修改 `LEFT(acct,1)` 的判断条件即可适配。

## 示例 3 ⭐⭐⭐ — 预算 vs 实际对比分析

假设预算表在 `Budget!A2:C50`（科目、预算金额、部门）。

```excel
=LET(
  /* 实际数据 */
  txn,      A2:G1000,
  acct,     INDEX(txn,,3),
  debit,    INDEX(txn,,5),
  credit,   INDEX(txn,,6),
  dept_col, INDEX(txn,,7),

  /* 预算数据 */
  bgt_acct, Budget!A2:A50,
  bgt_amt,  Budget!B2:B50,
  bgt_dept, Budget!C2:C50,

  /* 按科目+部门汇总实际发生额 */
  keys,     UNIQUE(HSTACK(FILTER(acct, acct<>""), FILTER(dept_col, acct<>""))),
  u_acct,   INDEX(keys,,1),
  u_dept,   INDEX(keys,,2),

  actual,   MAP(u_acct, u_dept, LAMBDA(a, d,
    SUMPRODUCT((acct=a) * (dept_col=d) * IF(ISNUMBER(debit), debit, 0)) -
    SUMPRODUCT((acct=a) * (dept_col=d) * IF(ISNUMBER(credit), credit, 0))
  )),
  budget,   MAP(u_acct, u_dept, LAMBDA(a, d,
    IFERROR(SUMPRODUCT((bgt_acct=a) * (bgt_dept=d) * bgt_amt), 0)
  )),
  variance, actual - budget,
  var_pct,  MAP(actual, budget, LAMBDA(a, b, IF(b=0, "-", ROUND((a-b)/ABS(b)*100, 1) & "%"))),

  header, {"科目","部门","预算","实际","差异","差异%"},
  body,   HSTACK(u_acct, u_dept, budget, actual, variance, var_pct),
  VSTACK(header, SORT(body, 5, -1))
)
```

**说明**：

1. 从交易明细汇总"科目+部门"维度的实际发生额
2. 用 `MAP` 对照预算表查找对应预算金额
3. 计算差异额和差异百分比，按差异降序排列

> 备选：添加 `FILTER(body, ABS(variance) > threshold)` 可只输出超预算的科目。

## 示例 4 ⭐⭐⭐⭐ — 现金流量追踪（含运行余额与预警）

```excel
=LET(
  data,     A2:G1000,
  dates,    INDEX(data,,1),
  acct,     INDEX(data,,3),
  memo,     INDEX(data,,4),
  debit,    INDEX(data,,5),
  credit,   INDEX(data,,6),

  /* 筛选现金类科目（1001 / 1002 等） */
  is_cash,  (LEFT(acct,4)="1001") + (LEFT(acct,4)="1002"),
  cash_txn, FILTER(data, is_cash, "无现金交易"),

  /* 计算每笔净变动 */
  c_debit,  INDEX(cash_txn,,5),
  c_credit, INDEX(cash_txn,,6),
  net,      MAP(c_debit, c_credit, LAMBDA(d, c,
              IF(ISNUMBER(d), d, 0) - IF(ISNUMBER(c), c, 0)
            )),

  /* 按日期排序 */
  sorted_idx, SORTBY(SEQUENCE(ROWS(cash_txn)), INDEX(cash_txn,,1), 1),
  sorted_net, INDEX(net, sorted_idx),
  sorted_dt,  INDEX(INDEX(cash_txn,,1), sorted_idx),
  sorted_memo, INDEX(INDEX(cash_txn,,4), sorted_idx),

  /* 运行余额 */
  init_bal,   50000,
  running,    SCAN(init_bal, sorted_net, LAMBDA(bal, x, bal + x)),

  /* 预警标记 */
  alert,      MAP(running, LAMBDA(b, IF(b < 10000, "⚠️低于警戒线", ""))),

  header, {"日期","摘要","净变动","余额","预警"},
  body,   HSTACK(sorted_dt, sorted_memo, sorted_net, running, alert),
  VSTACK(header, body)
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| 筛选 | 只保留现金类科目 | FILTER + LEFT |
| 净变动 | 借方 − 贷方 | MAP |
| 时间排序 | 按交易日期升序 | SORTBY + INDEX |
| 运行余额 | 逐笔累加 | SCAN |
| 预警 | 余额低于警戒线标记 | MAP + IF |

> 备选：修改 `init_bal` 和警戒线阈值 `10000` 即可适配不同企业场景。

## 示例 5 ⭐⭐⭐⭐⭐ — 完整损益表自动生成

假设科目表在 `COA!A2:C100`（科目编号、科目名称、类别：收入/成本/费用），Z1 为年份选择，Z2 为月份选择。

```excel
=LET(
  /* ① 数据源 */
  txn,     A2:G1000,
  t_date,  INDEX(txn,,1),
  t_acct,  INDEX(txn,,3),
  t_debit, INDEX(txn,,5),
  t_credit,INDEX(txn,,6),

  coa_id,   COA!A2:A100,
  coa_name, COA!B2:B100,
  coa_type, COA!C2:C100,

  /* ② 期间筛选 */
  yr,   Z1,
  mo,   Z2,
  mask, (YEAR(t_date)=yr) * (MONTH(t_date)=mo) * (t_acct<>""),

  /* ③ 按科目汇总 */
  u_acct,   SORT(UNIQUE(FILTER(t_acct, mask))),
  amounts,  MAP(u_acct, LAMBDA(a,
              SUMPRODUCT((t_acct=a) * mask * IF(ISNUMBER(t_credit), t_credit, 0)) -
              SUMPRODUCT((t_acct=a) * mask * IF(ISNUMBER(t_debit), t_debit, 0))
            )),
  names,    MAP(u_acct, LAMBDA(a, XLOOKUP(a, coa_id, coa_name, a))),
  types,    MAP(u_acct, LAMBDA(a, XLOOKUP(a, coa_id, coa_type, "其他"))),

  /* ④ 分类汇总 */
  rev_mask,  types = "收入",
  cost_mask, types = "成本",
  exp_mask,  types = "费用",

  rev_detail,  IFERROR(FILTER(HSTACK(names, amounts), rev_mask), {"（无收入科目）", 0}),
  cost_detail, IFERROR(FILTER(HSTACK(names, amounts), cost_mask), {"（无成本科目）", 0}),
  exp_detail,  IFERROR(FILTER(HSTACK(names, amounts), exp_mask), {"（无费用科目）", 0}),

  total_rev,   SUMPRODUCT(amounts * rev_mask),
  total_cost,  SUMPRODUCT(amounts * cost_mask),
  total_exp,   SUMPRODUCT(amounts * exp_mask),
  gross_profit, total_rev - total_cost,
  net_profit,  gross_profit - total_exp,

  /* ⑤ 组装损益表 */
  report, VSTACK(
    {"损益表 — " & yr & "年" & mo & "月", ""},
    {"",""},
    {"一、营业收入", ""},
    rev_detail,
    {"收入小计", total_rev},
    {"",""},
    {"二、营业成本", ""},
    cost_detail,
    {"成本小计", total_cost},
    {"",""},
    {"三、毛利润", gross_profit},
    {"",""},
    {"四、期间费用", ""},
    exp_detail,
    {"费用小计", total_exp},
    {"",""},
    {"═══════════════", "══════"},
    {"五、净利润", net_profit}
  ),

  report
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| ① | 加载交易和科目表 | 工作表引用 |
| ② | 按年/月动态筛选 | YEAR + MONTH + mask |
| ③ | 按科目汇总 + 查找科目名称 | UNIQUE + MAP + XLOOKUP |
| ④ | 按收入/成本/费用分类 | FILTER + SUMPRODUCT |
| ⑤ | 组装标准损益表格式 | VSTACK + HSTACK |

一个公式输出完整格式的损益表——切换年月即刻刷新全部数据，无需 VBA、无需数据透视表。

## 应用注意点

| 要点 | 说明 |
|------|------|
| 科目编码规则 | 示例使用首位字符区分类别，实际应用中建议维护独立的科目表（COA） |
| 借贷方向 | 收入类科目通常贷方为正，费用类科目通常借方为正——取决于会计准则 |
| 期间控制 | 使用年月下拉框控制期间范围，支持即时切换查看不同月份 |
| 精度 | 财务数据建议全程使用 `ROUND(..., 2)` 保留两位小数，避免浮点误差 |
| 审计追踪 | 可将中间变量 `u_acct` 和 `amounts` 单独输出用于核对 |
| 大数据性能 | 交易量 >10,000 条时，SUMPRODUCT 的性能优于 MAP+FILTER 嵌套 |
