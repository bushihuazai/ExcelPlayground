# 04 — BYROW 按行计算：将每一行压缩为一个值

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

`BYROW` 将二维区域的**每一行**作为一个一维数组传入 LAMBDA，返回单个值。结果是一个 N×1 的列向量。

语法：`=BYROW(array, LAMBDA(row, 表达式))`

---

## 示例 1 ⭐ — 每行求和

```excel
=BYROW(B2:F10, LAMBDA(r, SUM(r)))
```

**说明**：将 B–F 列每一行的值求和，返回 9 行 × 1 列的结果。等价于在辅助列写 `=SUM(B2:F2)` 再下拉。

---

## 示例 2 ⭐⭐ — 每行取最大值与最小值之差（极差）

```excel
=BYROW(B2:F50, LAMBDA(r, MAX(r) - MIN(r)))
```

**说明**：一行公式即可计算每行数据的波动幅度，常用于质量控制或成绩分析。

---

## 示例 3 ⭐⭐⭐ — 每行非空计数

```excel
=BYROW(B2:G100, LAMBDA(r,
  COUNTA(FILTER(r, r<>"", ""))
))
```

**说明**：`FILTER` 先剔除空值，`COUNTA` 再统计。由于 `FILTER` 全部为空时会报错，第三参数 `""` 作为空结果的默认值。

> ⚠️ 当 `FILTER` 结果为空字符串 `""` 时，`COUNTA` 会返回 1。更精确的写法是：
> ```excel
> =BYROW(B2:G100, LAMBDA(r, SUM(IF(r<>"",1,0))))
> ```

---

## 示例 4 ⭐⭐⭐⭐ — 每行拼接非空文本

```excel
=BYROW(A2:E50, LAMBDA(r,
  TEXTJOIN(", ", TRUE, r)
))
```

**说明**：`TEXTJOIN` 的第二参数 `TRUE` 表示忽略空值。这样每一行的非空单元格会被逗号拼接成一段文字。

---

## 示例 5 ⭐⭐⭐⭐⭐ — 每行加权平均（权重在固定行）

假设第 1 行（B1:F1）存放权重，B2:F50 是数据矩阵。

```excel
=BYROW(B2:F50, LAMBDA(r,
  LET(
    w, B1:F1,
    SUMPRODUCT(r, w) / SUM(w)
  )
))
```

**说明**：

- `r` 是当前行的数据（一维数组）
- `w` 是权重行（固定引用）
- `SUMPRODUCT` 进行点乘求和后除以权重之和

适合教学评分、指标综合打分等场景。

---

## 应用注意点

| 要点 | 说明 |
|------|------|
| 返回值形状 | LAMBDA 必须返回**单个值**（标量），不能返回数组 |
| 引用外部区域 | LAMBDA 内部可以引用外部固定区域（如权重行），不限于传入的行 |
| 与 MAP 区别 | MAP 处理每个**单元格**；BYROW 处理每**行**（可含多列） |
| 与 BYCOL 区别 | BYROW 遍历行，返回列向量；BYCOL 遍历列，返回行向量 |
| 性能 | BYROW 底层逐行调用 LAMBDA，大数据量时可能不如原生公式快 |
