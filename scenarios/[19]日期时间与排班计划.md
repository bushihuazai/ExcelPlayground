# 19 — 日期时间与排班计划：工作日、日历与班次的纯公式方案

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

日期与时间计算是 Excel 最高频的应用之一：工作日推算、假期排除、月历/年历生成、轮班排配等。本场景综合运用 `SEQUENCE`、`INDEX`、`MAP`、`FILTER`、`LET`、`MAKEARRAY`、`HSTACK`、`VSTACK` 等函数，用纯公式实现日历和排班自动化。

---

## 示例 1 ⭐ — 生成月度日历表

```excel
=LET(
  year,      2025,
  month,     4,
  first_day, DATE(year, month, 1),
  days,      DAY(EOMONTH(first_day, 0)),
  all_dates, SEQUENCE(days, 1, first_day, 1),

  /* 星期标题（周一=1 到 周日=7） */
  weekday_num, WEEKDAY(all_dates, 2),
  offset,      WEEKDAY(first_day, 2) - 1,
  total_cells, days + offset,
  rows,        INT((total_cells - 1) / 7) + 1,

  /* 填入 7列 × N行 的网格 */
  grid, MAKEARRAY(rows, 7, LAMBDA(r, c,
    LET(
      pos, (r - 1) * 7 + c - offset,
      IF(AND(pos >= 1, pos <= days),
        DAY(first_day + pos - 1),
        ""
      )
    )
  )),

  header, {"一","二","三","四","五","六","日"},
  VSTACK(year & "年" & month & "月", header, grid)
)
```

**说明**：

1. `EOMONTH(first_day, 0)` 获取月末日期，从而计算当月天数
2. `WEEKDAY(..., 2)` 返回 ISO 星期编号（周一=1）
3. `MAKEARRAY` 根据行列位置填入对应日期号或留空

> 备选：将 `year` 和 `month` 改为单元格引用即可做成交互式月历。

## 示例 2 ⭐⭐ — 工作日计算（排除周末和节假日）

假设法定假日清单在 `Holiday!A2:A30`。

```excel
=LET(
  start,     DATE(2025, 1, 1),
  end_date,  DATE(2025, 12, 31),
  holidays,  Holiday!A2:A30,

  /* 生成日期序列 */
  n_days,    end_date - start + 1,
  all_dates, SEQUENCE(n_days, 1, start, 1),
  wd,        WEEKDAY(all_dates, 2),

  /* 工作日条件：非周末 + 非假日 */
  is_weekday, (wd <= 5),
  is_holiday, MAP(all_dates, LAMBDA(d,
    SUM((holidays = d) * 1) > 0
  )),
  is_work,   is_weekday * NOT(is_holiday),

  workdays,  FILTER(all_dates, is_work),
  total,     ROWS(workdays),

  VSTACK(
    {"工作日总数", total},
    {"",""},
    {"日期列表（前 20 天）","星期"},
    HSTACK(
      TAKE(workdays, MIN(20, total)),
      MAP(TAKE(workdays, MIN(20, total)), LAMBDA(d,
        CHOOSE(WEEKDAY(d, 2), "周一","周二","周三","周四","周五","周六","周日")
      ))
    )
  )
)
```

**说明**：

1. `SEQUENCE` 生成全年日期序列
2. `WEEKDAY(..., 2) <= 5` 判断是否为工作日（周一到周五）
3. `MAP + SUM` 检查每个日期是否在假日列表中
4. 两个条件组合筛选出所有工作日

> 备选：用 `NETWORKDAYS(start, end_date, holidays)` 可快速得到工作日总数，但本方法可获取完整日期列表。

## 示例 3 ⭐⭐⭐ — 项目工期与里程碑甘特图（文本版）

假设任务列表在 `A2:D10`（任务名、开始日期、结束日期、负责人）。

```excel
=LET(
  tasks,     A2:D10,
  t_name,    INDEX(tasks,,1),
  t_start,   INDEX(tasks,,2),
  t_end,     INDEX(tasks,,3),
  t_owner,   INDEX(tasks,,4),
  n,         ROWS(tasks),

  /* 项目时间范围（按周） */
  proj_start, MIN(t_start),
  proj_end,   MAX(t_end),
  n_weeks,    INT((proj_end - proj_start) / 7) + 1,
  week_starts, SEQUENCE(1, n_weeks, proj_start, 7),

  /* 甘特条：每任务 × 每周 */
  gantt, MAKEARRAY(n, n_weeks, LAMBDA(r, c,
    LET(
      ws, INDEX(week_starts,, c),
      we, ws + 6,
      ts, INDEX(t_start, r),
      te, INDEX(t_end, r),
      IF(AND(te >= ws, ts <= we), "██", "")
    )
  )),

  /* 周标签 */
  week_labels, MAP(week_starts, LAMBDA(d, TEXT(d, "MM/DD"))),

  /* 任务信息列 */
  info, HSTACK(t_name, t_owner,
    MAP(t_start, t_end, LAMBDA(s, e, e - s + 1 & "天"))
  ),

  /* 组装 */
  header, HSTACK("任务", "负责人", "工期", week_labels),
  body,   HSTACK(info, gantt),
  VSTACK(header, body)
)
```

**说明**：

1. 计算项目整体的时间跨度，按周切分为列
2. `MAKEARRAY` 判断每个任务在每周是否"正在进行"，是则填充 `██`
3. 输出包含任务名、负责人、工期天数和可视化甘特条的完整表格

> 备选：将 `7`（周粒度）改为 `1`（天粒度）可生成更精细的甘特图，但列数较多。

## 示例 4 ⭐⭐⭐⭐ — 三班倒轮班自动排配

假设员工名单在 `E2:E20`，需要为未来 N 天自动生成三班轮换计划。

```excel
=LET(
  names,     FILTER(E2:E20, E2:E20<>""),
  n_people,  ROWS(names),
  start_dt,  DATE(2025, 5, 1),
  n_days,    30,
  shifts,    {"早班","中班","晚班"},
  n_shifts,  3,

  /* 日期序列 */
  dates,     SEQUENCE(n_days, 1, start_dt, 1),
  weekdays,  MAP(dates, LAMBDA(d,
    CHOOSE(WEEKDAY(d, 2), "一","二","三","四","五","六","日")
  )),

  /* 排班矩阵：每人每天分配一个班次 */
  schedule, MAKEARRAY(n_days, n_people, LAMBDA(r, c,
    LET(
      /* 按天轮转：人员索引 + 天偏移 决定班次 */
      shift_idx, MOD(c - 1 + INT((r - 1) / n_people), n_shifts) + 1,
      INDEX(shifts,, shift_idx)
    )
  )),

  /* 每日各班次人数统计 */
  daily_check, MAP(dates, LAMBDA(d,
    LET(
      day_row, INDEX(schedule, MATCH(d, dates, 0), SEQUENCE(1, n_people)),
      TEXTJOIN("/", TRUE,
        MAP(shifts, LAMBDA(s, s & ":" & SUM((day_row=s)*1)))
      )
    )
  )),

  /* 组装 */
  header, HSTACK("日期", "星期", TOROW(names), "班次统计"),
  body,   HSTACK(dates, weekdays, schedule, daily_check),
  VSTACK(header, body)
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| 日期生成 | 连续 N 天 | SEQUENCE |
| 星期标注 | 中文星期 | MAP + CHOOSE + WEEKDAY |
| 排班分配 | 员工 × 天数的轮转矩阵 | MAKEARRAY + MOD |
| 班次统计 | 每天各班次人数核验 | MAP + TEXTJOIN |

> 备选：修改 `shifts` 常量即可扩展为四班三运转、两班倒等其他排班模式。
> 备选：添加 `IF(WEEKDAY(d,2)>=6, "休", ...)` 可为周末自动标注休息。

## 示例 5 ⭐⭐⭐⭐⭐ — 考勤汇总仪表板（出勤 / 加班 / 请假）

假设考勤记录在 `Attend!A2:F5000`（日期、工号、姓名、上班打卡、下班打卡、请假类型），员工名单在 `E2:F50`（工号、姓名），Z1 为月份选择。

```excel
=LET(
  /* ① 数据源 */
  att_dt,    Attend!A2:A5000,
  att_id,    Attend!B2:B5000,
  att_name,  Attend!C2:C5000,
  att_in,    Attend!D2:D5000,
  att_out,   Attend!E2:E5000,
  att_leave, Attend!F2:F5000,

  emp_id,    E2:E50,
  emp_name,  F2:F50,

  /* ② 月份筛选 */
  yr,   YEAR(Z1),
  mo,   MONTH(Z1),
  mask, (YEAR(att_dt)=yr) * (MONTH(att_dt)=mo) * (att_id<>""),
  n_workdays, NETWORKDAYS(DATE(yr, mo, 1), EOMONTH(DATE(yr, mo, 1), 0)),

  /* ③ 按员工汇总 */
  u_id,  FILTER(emp_id, emp_id<>""),
  u_name, MAP(u_id, LAMBDA(id, XLOOKUP(id, emp_id, emp_name, id))),

  /* 出勤天数 */
  attend_days, MAP(u_id, LAMBDA(id,
    SUMPRODUCT((att_id=id) * mask * (att_leave="") * 1)
  )),

  /* 加班时长（下班打卡 - 18:00，超出部分按小时计） */
  overtime_hrs, MAP(u_id, LAMBDA(id,
    ROUND(SUMPRODUCT(
      (att_id=id) * mask * (att_leave="") *
      MAX(0, (att_out - TIME(18,0,0)) * 24)
    ), 1)
  )),

  /* 请假天数（按类型） */
  sick_days, MAP(u_id, LAMBDA(id,
    SUMPRODUCT((att_id=id) * mask * (att_leave="病假") * 1)
  )),
  annual_days, MAP(u_id, LAMBDA(id,
    SUMPRODUCT((att_id=id) * mask * (att_leave="年假") * 1)
  )),
  other_leave, MAP(u_id, LAMBDA(id,
    SUMPRODUCT((att_id=id) * mask * (att_leave<>"") * (att_leave<>"病假") * (att_leave<>"年假") * 1)
  )),

  /* 出勤率 */
  rate, MAP(attend_days, LAMBDA(a, ROUND(a / MAX(n_workdays, 1) * 100, 1) & "%")),

  /* ④ 异常标记 */
  alert, MAP(attend_days, sick_days, LAMBDA(a, s,
    IF(a < n_workdays * 0.8, "⚠️出勤不足",
      IF(s >= 5, "⚠️病假较多", ""))
  )),

  /* ⑤ 组装 */
  header, {"工号","姓名","出勤天数","加班时长","病假","年假","其他假","出勤率","备注"},
  body,   HSTACK(u_id, u_name, attend_days, overtime_hrs, sick_days, annual_days, other_leave, rate, alert),

  VSTACK(
    {"考勤汇总 — " & yr & "年" & mo & "月", "","","","","","","",""},
    HSTACK("当月工作日", n_workdays, "","","","","","",""),
    {"","","","","","","","",""},
    header,
    SORT(body, 3, -1)
  )
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| ① | 加载考勤记录和员工表 | 工作表引用 |
| ② | 月份筛选 + 工作日天数 | YEAR/MONTH + NETWORKDAYS |
| ③ | 逐员工统计各项指标 | MAP + SUMPRODUCT |
| ④ | 异常标记 | MAP + IF |
| ⑤ | 组装带表头的汇总报表 | VSTACK + HSTACK + SORT |

一个公式输出完整月度考勤汇总——切换月份即刻刷新所有员工的出勤、加班、请假及异常标记。

## 应用注意点

| 要点 | 说明 |
|------|------|
| 日期序列号 | 确保日期是 Excel 日期序列号而非文本，否则比较运算会失效 |
| 时间格式 | 打卡时间应为 TIME 格式（0~1 之间的小数），`TIME(18,0,0)` = 0.75 |
| 假日维护 | 法定假日和调休需手动维护列表或从外部导入 |
| 排班公平性 | MOD 轮转保证每人均匀轮转，但长期使用需定期检查夜班分配是否均衡 |
| NETWORKDAYS | 内置函数自动排除周末，传入第三参数可同时排除假日 |
| 扩展 | 考勤统计可封装为命名函数 `ATTEND_SUMMARY(emp_id, month)`，便于跨表调用 |
