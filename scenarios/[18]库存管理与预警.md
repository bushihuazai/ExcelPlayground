# 18 — 库存管理与预警：从台账到智能补货的纯公式方案

> **适用版本**：Microsoft 365 / Excel 2021+

## 简介

库存管理涉及库存水位监控、安全库存计算、ABC 分类、库龄分析、自动补货建议等。本场景综合运用 `LET`、`FILTER`、`SORT`、`UNIQUE`、`MAP`、`SCAN`、`HSTACK`、`VSTACK`、`XLOOKUP` 等函数，在纯公式中实现从库存台账到管理决策的完整链路。

假设库存主数据 `A2:H300`，列依次为：**SKU(A)、品名(B)、分类(C)、当前库存(D)、安全库存(E)、单价(F)、最近入库日(G)、月均出库量(H)**。

---

## 示例 1 ⭐ — 库存预警清单：低于安全库存的商品

```excel
=LET(
  data,    A2:H300,
  sku,     INDEX(data,,1),
  name,    INDEX(data,,2),
  stock,   INDEX(data,,4),
  safety,  INDEX(data,,5),
  low,     FILTER(
             HSTACK(sku, name, stock, safety, safety - stock),
             stock < safety,
             "所有商品库存充足 ✓"
           ),
  header,  {"SKU","品名","当前库存","安全库存","缺口数量"},
  VSTACK(header, SORT(low, 5, -1))
)
```

**说明**：筛选当前库存低于安全库存的商品，计算缺口数量并按缺口降序排列。这是最基本的库存预警逻辑。

## 示例 2 ⭐⭐ — 库存周转天数与补货建议

```excel
=LET(
  data,     A2:H300,
  sku,      INDEX(data,,1),
  name,     INDEX(data,,2),
  stock,    INDEX(data,,4),
  safety,   INDEX(data,,5),
  monthly,  INDEX(data,,8),
  daily,    monthly / 30,

  /* 库存可用天数 */
  days_left, MAP(stock, daily, LAMBDA(s, d,
    IF(d <= 0, 9999, ROUND(s / d, 0))
  )),

  /* 补货建议量 = 安全库存 + 30天需求 - 当前库存 */
  reorder,  MAP(safety, monthly, stock, LAMBDA(sf, m, s,
    MAX(0, sf + m - s)
  )),

  /* 紧急程度 */
  urgency,  MAP(days_left, LAMBDA(d,
    IF(d <= 7, "🔴紧急",
      IF(d <= 14, "🟡关注",
        IF(d <= 30, "🟢正常", "充足")))
  )),

  header, {"SKU","品名","当前库存","可用天数","补货建议量","紧急程度"},
  body,   HSTACK(sku, name, stock, days_left, reorder, urgency),
  result, FILTER(body, INDEX(body,,5) > 0, "无需补货"),
  VSTACK(header, SORT(result, 4, 1))
)
```

**说明**：

1. `stock / daily` 计算当前库存可支撑的天数
2. 补货建议量 = 安全库存 + 月需求 − 当前库存（负值截断为 0）
3. 按可用天数升序排列，紧急的排在前面

> 备选：将 `30天需求` 调整为 `lead_time_days * daily` 可引入采购提前期计算。

## 示例 3 ⭐⭐⭐ — ABC 分类分析（帕累托法则）

```excel
=LET(
  data,    A2:H300,
  sku,     INDEX(data,,1),
  name,    INDEX(data,,2),
  stock,   INDEX(data,,4),
  price,   INDEX(data,,6),
  monthly, INDEX(data,,8),

  /* 月消耗金额 */
  value,     monthly * price,
  total_val, SUM(value),

  /* 按消耗金额降序排序 */
  sorted_idx, SORTBY(SEQUENCE(ROWS(data)), value, -1),
  s_sku,     INDEX(sku, sorted_idx),
  s_name,    INDEX(name, sorted_idx),
  s_value,   INDEX(value, sorted_idx),

  /* 累计占比 */
  cum_val,   SCAN(0, s_value, LAMBDA(acc, v, acc + v)),
  cum_pct,   ROUND(cum_val / total_val * 100, 1),

  /* ABC 分类 */
  abc_class, MAP(cum_pct, LAMBDA(p,
    IF(p <= 80, "A",
      IF(p <= 95, "B", "C"))
  )),

  header, {"SKU","品名","月消耗额","累计%","分类"},
  body,   HSTACK(s_sku, s_name, ROUND(s_value, 0), cum_pct & "%", abc_class),
  VSTACK(header, body)
)
```

**说明**：

1. 月消耗额 = 月均出库量 × 单价
2. 按消耗额降序排列，用 `SCAN` 计算累计值
3. 累计占比 ≤80% 为 A 类（高价值），80%~95% 为 B 类，>95% 为 C 类

> 备选：修改 `80` 和 `95` 的阈值可自定义分类标准。ABC 分类是库存管理中优先级决策的基础。

## 示例 4 ⭐⭐⭐⭐ — 库龄分析报告（按入库时间分段）

```excel
=LET(
  data,     A2:H300,
  sku,      INDEX(data,,1),
  name,     INDEX(data,,2),
  category, INDEX(data,,3),
  stock,    INDEX(data,,4),
  price,    INDEX(data,,6),
  last_in,  INDEX(data,,7),
  today,    TODAY(),

  /* 库龄（天） */
  age_days, today - last_in,

  /* 分段定义 */
  breaks,   {0, 30, 90, 180, 365, 9999},
  n,        COLUMNS(breaks) - 1,
  labels,   MAP(SEQUENCE(1, n), LAMBDA(i,
    IF(INDEX(breaks,,i+1) = 9999,
      INDEX(breaks,,i) & "天以上",
      INDEX(breaks,,i) & "~" & INDEX(breaks,,i+1) & "天"
    )
  )),

  /* 按分类 × 库龄段汇总库存金额 */
  u_cat,    SORT(UNIQUE(FILTER(category, category<>""))),
  n_cat,    ROWS(u_cat),

  matrix,   MAKEARRAY(n_cat, n, LAMBDA(r, c,
    LET(
      cat, INDEX(u_cat, r),
      lo,  INDEX(breaks,,c),
      hi,  INDEX(breaks,,c+1),
      ROUND(SUMPRODUCT(
        (category=cat) * (age_days>=lo) * (age_days<hi) * stock * price
      ), 0)
    )
  )),

  /* 行合计 */
  row_total, BYROW(matrix, LAMBDA(r, SUM(r))),

  /* 组装 */
  header,   HSTACK("分类", labels, "合计"),
  body,     HSTACK(u_cat, matrix, row_total),

  /* 合计行 */
  col_total, HSTACK("合计", BYCOL(matrix, LAMBDA(c, SUM(c))), SUM(matrix)),

  VSTACK(header, body, col_total)
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| 库龄计算 | TODAY() - 最近入库日 | 日期差 |
| 分段定义 | {0,30,90,180,365,9999} | 常量数组 |
| 交叉汇总 | 分类 × 库龄段的库存金额 | MAKEARRAY + SUMPRODUCT |
| 行列合计 | 各维度小计 | BYROW + BYCOL + SUM |

> 备选：将 `stock * price` 替换为 `stock` 可切换为按数量统计的库龄报告。

## 示例 5 ⭐⭐⭐⭐⭐ — 综合库存管理仪表板

假设 Z1 为分类下拉框（含"全部"），出入库流水在 `Flow!A2:E5000`（日期、SKU、类型:入库/出库、数量、备注）。

```excel
=LET(
  /* ① 主数据 + 动态筛选 */
  master,   A2:H300,
  m_sku,    INDEX(master,,1),
  m_name,   INDEX(master,,2),
  m_cat,    INDEX(master,,3),
  m_stock,  INDEX(master,,4),
  m_safety, INDEX(master,,5),
  m_price,  INDEX(master,,6),
  m_lastin, INDEX(master,,7),
  m_monthly,INDEX(master,,8),

  cat_filter, IF(Z1="全部", TRUE, m_cat=Z1),
  sel,      FILTER(master, cat_filter * (m_sku<>"")),

  /* ② 汇总 KPI */
  s_stock,   INDEX(sel,,4),
  s_safety,  INDEX(sel,,5),
  s_price,   INDEX(sel,,6),
  s_monthly, INDEX(sel,,8),
  sku_count, ROWS(sel),
  total_val, ROUND(SUM(s_stock * s_price), 0),
  alert_cnt, SUM((s_stock < s_safety) * 1),
  avg_turn,  ROUND(SUM(s_monthly * s_price) * 12 / MAX(total_val, 1), 1),

  /* ③ 近 30 天出库趋势 */
  flow_dt,   Flow!A2:A5000,
  flow_sku,  Flow!B2:B5000,
  flow_type, Flow!C2:C5000,
  flow_qty,  Flow!D2:D5000,
  today,     TODAY(),
  last30,    SEQUENCE(30, 1, today - 29, 1),
  daily_out, MAP(last30, LAMBDA(d,
    SUMPRODUCT(
      (flow_dt = d) * (flow_type = "出库") *
      IF(Z1="全部", 1, (XLOOKUP(flow_sku, m_sku, m_cat, "")=Z1)) *
      IF(ISNUMBER(flow_qty), flow_qty, 0)
    )
  )),

  /* ④ ABC 分类占比 */
  s_val,       s_monthly * s_price,
  sorted_val,  SORT(s_val, , -1),
  cum,         SCAN(0, sorted_val, LAMBDA(a, v, a + v)),
  total_s,     SUM(s_val),
  a_count,     SUM((cum / total_s <= 0.8) * 1),
  b_count,     SUM((cum / total_s > 0.8) * (cum / total_s <= 0.95) * 1),
  c_count,     sku_count - a_count - b_count,

  /* ⑤ 组装仪表板 */
  report, VSTACK(
    {"【库存总览】", ""},
    HSTACK("SKU 总数", sku_count),
    HSTACK("库存总金额", total_val),
    HSTACK("预警商品数", alert_cnt),
    HSTACK("年库存周转率", avg_turn),
    {"",""},
    {"【ABC 分类分布】", "SKU数"},
    HSTACK("A 类（核心）", a_count),
    HSTACK("B 类（重要）", b_count),
    HSTACK("C 类（一般）", c_count),
    {"",""},
    {"【近30天每日出库量】", ""},
    HSTACK(last30, daily_out)
  ),

  report
)
```

**说明**：

| 步骤 | 功能 | 核心函数 |
|------|------|----------|
| ① | 主数据加载与分类筛选 | FILTER + IF |
| ② | KPI 汇总（SKU数、总金额、预警数、周转率） | SUM、ROWS、ROUND |
| ③ | 近 30 天出库趋势 | SEQUENCE + MAP + SUMPRODUCT |
| ④ | ABC 分类占比 | SORT + SCAN + 累计百分比 |
| ⑤ | 组装仪表板 | VSTACK + HSTACK |

一个公式输出完整的库存管理仪表板——切换分类即刻刷新全部指标和趋势。

---

## 应用注意点

| 要点 | 说明 |
|------|------|
| 数据实时性 | 库存数量应保持实时更新；如有出入库流水，可用 SUMPRODUCT 动态计算当前库存 |
| 安全库存设定 | 安全库存 = 采购提前期 × 日均消耗 × 安全系数，建议单独计算后填入 |
| ABC 分类阈值 | 80/95 是经典帕累托阈值，不同行业可调整 |
| 库龄精度 | "最近入库日"仅反映最后一批，精确库龄需要 FIFO 逻辑（更复杂） |
| 性能优化 | SKU 超 1,000 或流水超 50,000 行时，SUMPRODUCT 比 MAP+FILTER 更高效 |
| 扩展 | 可将预警逻辑封装为命名函数 `STOCK_ALERT(sku)`，方便在其他报表中复用 |
